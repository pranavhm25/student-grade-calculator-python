# Student Grade Calculator

A simple Python program that calculates the average marks, grade, and pass/fail status for multiple students based on their subject marks.

---

## Features

- Input marks for multiple students
- Supports multiple subjects per student
- Calculates average marks
- Assigns grades automatically
- Displays pass/fail status
- Stores student data in a list

---

## Technologies Used

- Python
- Basic programming concepts

---

## Concepts Practiced

- Functions
- Loops
- Lists
- Dictionaries
- Conditional statements
- User input/output

---

## How to Run the Program

1. Clone the repository
```bash
git clone https://github.com/pranavhm25/student-grade-calculator-python.git
```

2. Navigate to the project folder
```bash
cd student-grade-calculator
```

3. Run the program
```bash
python main.py
```
